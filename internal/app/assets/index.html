云平台运维台账

使用本地账号登录后进入运维台账和开发者工作台。

开发种子账号需显式启用并设置本地密码；公开部署请接入正式身份源。

首次部署初始化

数据库表已自动初始化。这里会创建第一个平台管理员账号，不会生成默认弱口令。

云平台运维台账

按云平台、账号和资源类型管理运维现场。

资产总览

项目 -> 云平台 -> 账号 -> Zone / 类型 -> 资产

资产树 按账号：云平台 -> 账号 -> 项目 -> 类型；按项目：项目 -> 云平台 -> 账号 -> 类型
资产详情 查看规格、同步、巡检与变更概况

开发者工作台

按环境查看工具入口、账号密码、临时凭证和 EC2 WebSSH。

应用入口

当前环境的业务应用入口，默认展开。

工具入口

全局运维工具入口，默认展开。

资产列表

WebSSH 目标为 AWS EC2，敏感环境进入审批。

审计工作台

按操作时间追踪登录、权限、审批、凭证、WebSSH、同步和告警处理留痕。

最近 50 条随首页加载，接口最多返回 100 条

审计日志

每条记录包含操作时间、操作人、动作、对象、结果、来源和摘要。

审计范围

当前已覆盖关键安全动作。

  • 身份登录、退出、越权拒绝
  • 审批申请提交、审批处理、短期授权
  • 凭证查看明文、复制凭证、保存凭证
  • 访问WebSSH 打开、会话授权校验
  • 运维云同步、费用同步、告警处理、附件上传

提交审批申请

审批与凭证

审批待办

新增资产

支持手工维护非云发现资产。

批量编辑

按当前资产树筛选条件批量更新资产治理字段；留空字段不会修改。

新增工具资产

工具由运维配置，同时作为 tool 类资产进入资产树和开发者工作台。

工具资产只维护入口和凭证策略；WebSSH 请在 AWS EC2 资产上申请。

凭证管理

给工具、资产或云账号维护受控凭证。保存后只展示脱敏值，查看和复制都会审计。

标签治理

基于台账资产和云资源 tag 值识别 Project / Environment 缺口。

项目分账

只读估算：按云账号内非 stale 资产项目占比分摊账号费用。

权限管理

配置用户、角色、角色权限和审批流程。

角色配置

定义角色编码、名称和层级。

角色权限

按角色、环境、资源范围和动作配置权限。

审批流程

拖动步骤调整顺序，保存后作为审批编排定义。

新增变更

记录执行窗口、风险级别和回滚方案。

自动巡检

巡检由后台自动拨测、同步检查和资产规则生成；这里仅保留当前口径说明。

告警记录

自动拨测和巡检规则生成，恢复后可自动关闭；未恢复的告警由运维处理留痕。
